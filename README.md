# VisualHabitTracker
The Visual Habit Tracker is a web application designed to help users cultivate consistency through clear, visual feedback. Unlike traditional text-based lists, this system utilizes a grid-based interface to display habit patterns, streaks, and long-term progress.

✨ Key Features

    1. Habit ManagementCreation: Users can easily create and manage multiple daily habits such as reading, exercise, or coding practice.Tracking: Mark habits as completed by simply clicking a calendar cell or checkbox.
    
    2. Heatmap VisualizationThe application uses a color-coded grid to reflect consistency:
           Gray: Habit not completed.
           Light Green: Occasionally completed.
           Dark Green: High consistency.
           
    3. Analytics and StreaksStreak Tracking: View current and longest streaks to stay motivated.Detailed Statistics: Monitor completion rates and total days completed for every habit.
